Freedom Project

Context

Throughout this year, we had to make our freedom project which is about an exisiting item and a future innovation based on that item.

Content

We had to make a website about a topic we are interested in using html and css. My project is based on market research so I researched about it and thought of an innovation idea for it. We also had to present our projects so we had to make google slides for it as well.

Reflection

Some challenges were trying to position the website. This is because I didn't want it to be a slob of text but when I tried to position everything, it just seems to look weird because of bad placement. I solved this by using rows and containers and it made my website much neater. A takeaway from all this is to start on the project earlier than later because It took me pretty late to finish the project.
